# MiniGames
小游戏仓库

代码提交规范：
1、寝室长负责同步master_dev到master_dev_自己的寝室号
2、寝室长负责发起合并请求master_dev_自己的寝室号到master_dev
3、成员在自己的分支上可以自由提交代码或拉取代码【git push 和 git pull】
4、成员可以自由同步master_dev_自己的寝室号到自己的分支
5、成员上传代码至master_dev的过程：
5.1、master_dev_自己的寝室号 -> 自己的远程分支【同步分支】
5.2、git pull
5.3、git push【假如本地已经提交过，则不需要】
5.4、发起合并请求：自己的远程分支 -> master_dev_自己的寝室号
